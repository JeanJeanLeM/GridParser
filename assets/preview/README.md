# Style preview images

**By style (apple sample):** The prompt builder uses these SVGs so the preview updates when you change the Style dropdown. Included:

- `apple-minimalist-vector.svg`
- `apple-watercolor.svg`
- `apple-realistic.svg`
- `apple-cartoon.svg`
- `apple-sketch.svg`

**By palette (optional):** You can add PNGs per palette for palette-driven preview; the app falls back to style-based images if needed.

- `apple-flat-and-vivid.png`
- `apple-soft-and-pastel.png`
- `apple-monochrome.png`
- `apple-natural.png`
